  <how-to-verify>
Build and launch the app, then verify each success criterion:

**DEST-02: Mounted Volume destination**
1. Open Settings → Destinations tab
2. Click "+" → choose "Mounted Volume"
3. Select any external drive or NAS mounted in /Volumes
4. Click "Test Connection" — confirm green "Connection successful"
5. Click "Save" — destination appears in list with green dot
6. Trigger a manual backup ("Back Up Now" in menu bar)
7. Confirm files appear on the mounted volume at its rootPath

**DEST-04: iCloud Drive destination**
1. Settings → Destinations → "+" → "iCloud Drive"
2. Click "Choose…" → navigate to a folder inside iCloud Drive
3. Click "Test Write Access" — confirm success
4. Click "Save" — destination appears with green dot
5. Trigger manual backup
6. Confirm backup version folder appears in the chosen iCloud path

**DEST-06: Live destination status**
1. With at least one destination configured, confirm green dot in settings list
2. Confirm green dot for that destination also appears in menu bar popup
3. Disconnect or rename the destination path (simulate offline)
4. Wait up to 35 seconds for next polling cycle (or trigger probe manually via Retry Now)
5. Confirm dot turns red and shows offline label
6. Click the red dot row → confirm error detail sheet opens with error message and "Retry Now" button
7. Reconnect the destination path → click "Retry Now" → confirm dot returns to green

**DEST-07: Sleep/wake reconnection** (SMB only — can skip if no NAS available)
1. If NAS/SMB destination configured: put Mac to sleep (Apple menu → Sleep)
2. Wake the Mac
3. Wait 10-15 seconds for reconnection delay + network establishment
4. Confirm NAS destination dot returns to green without user action

**General: Remove destination**
1. Swipe-to-delete or select and click "-" on any destination
2. Confirm it disappears from the list
3. Trigger backup — confirm backup does NOT write to removed destination path

**If DEST-03 (SMB) is available to test:**
1. Settings → Destinations → "+" → "SMB / NFS Network Drive"
2. Enter Host, Share, Username, Password
3. Click "Test Connection" — confirm mount attempt (up to 10 seconds)
4. If successful: click "Save" — confirm destination appears with green dot
5. Trigger backup — confirm files appear on the SMB share
