<decisions_made>

- xxHash64 (not CryptoKit SHA256) used for checksums — xxHash-Swift confirmed available in 01-01
- COPYFILE_CLONE | COPYFILE_ALL (not FORCE) for ExFAT compatibility
- VersionManager as stateless enum (no stored properties) — all state in DatabasePool
- Write-then-cleanup: status=deleting in DB transaction before any FileManager.removeItem
- Corrupt versions excluded from retention count, never pruned, kept for user inspection
- Walk-and-collect pruning (not prefix-filter) — handles locked versions in pruning window correctly
- BackupVersion.totalBytes is Int (not Int64) — match actual model to avoid type mismatch in VersionManager.finalizeCopy
</decisions_made>
