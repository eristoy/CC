<!-- Key schema decisions — executors must implement these exactly. -->

Version lifecycle (GRDB status column — TEXT NOT NULL DEFAULT 'pending'):
  pending -> copying -> copy_complete -> verifying -> verified
                                                   -> corrupt
  (also: deleting — used by pruning write-then-cleanup pattern)

Only `verified` versions count toward retention limit or are eligible for pruning.
Only `verified` versions are eligible for restore (future phase).

Version ID format: "{ISO8601-ms}-{UUID-prefix-8}"
  e.g., "2026-02-25T143022.456-a3f8b12c"
  - Use millisecond precision (DateFormatter format: "yyyy-MM-dd'T'HH:mm:ss.SSS")
  - Append "-" + UUID().uuidString.prefix(8) to prevent collision on rapid consecutive backups

DestinationType in Phase 1: only `.local`
  Design the enum for extensibility: local | nas | icloud | github (Phase 4-6 will add cases)

retentionCount default: 10 (stored per destination, not per project)

GRDB version: 7.0.0+ (requires Swift 6.1+ / Xcode 16.3+)
Use DatabasePool (not DatabaseQueue) — WAL mode is automatic, enables concurrent reads.
</interfaces>

Create AppDatabase.swift:
```swift
import GRDB
import Foundation

public final class AppDatabase: Sendable {
    public let pool: DatabasePool

    public static func makeShared(at path: String) throws -> AppDatabase {
        let pool = try DatabasePool(path: path)
        // DatabasePool automatically enables WAL mode on open.
        let db = AppDatabase(pool: pool)
        try db.applyMigrations()
        return db
    }

    public static func makeInMemory() throws -> AppDatabase {
        let pool = try DatabasePool()  // in-memory for tests
        let db = AppDatabase(pool: pool)
        try db.applyMigrations()
        return db
    }

    private init(pool: DatabasePool) { self.pool = pool }

    private func applyMigrations() throws {
        var migrator = DatabaseMigrator()
        #if DEBUG
        migrator.eraseDatabaseOnSchemaChange = true
        #endif
        Schema.registerMigrations(in: &migrator)
        try migrator.migrate(pool)
    }
}
```
